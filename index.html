✨ Features

ฟีเจอร์ของ วิล เซอร์ฟอท (Bot)

ฟีเจอร์ระบบเพลง YouTube ครบจบในบอทเดียว 🎵

🎵

เล่นเพลง YouTube

เล่นเพลงจาก YouTube ได้ทันที รองรับทั้งค้นหาชื่อเพลงและวาง URL โดยตรง ใช้ yt-dlp ดึงเสียงคุณภาพสูง

📋

ระบบคิวเพลง

เพิ่มเพลงเข้าคิวได้เรื่อยๆ บอทจะเล่นต่อเนื่องอัตโนมัติ เมื่อเล่นหมดคิวบอทจะออกจากช่องเสียง

⏸️

ควบคุมการเล่น

หยุดชั่วคราว เล่นต่อ ข้ามเพลง และหยุดทั้งหมดได้ทุกคนในเซิร์ฟเวอร์ ไม่จำกัดสิทธิ์

🔗

รองรับ URL และค้นหา

วาง URL ของ YouTube หรือพิมพ์ชื่อเพลงก็ได้ ระบบค้นหาด้วย ytsearch อัตโนมัติ

🔄

เชื่อมต่อ Voice อัตโนมัติ

บอทเข้าช่อง Voice ของผู้ใช้อัตโนมัติเมื่อสั่งเล่นเพลง และย้ายช่องตามได้หากผู้ใช้เปลี่ยนห้อง

🎮 Commands

Slash Commands

คำสั่งทั้งหมดที่ใช้งานได้บน Discord 🎀

🎵 ระบบเพลง
▶️
/play
เล่นเพลงจาก YouTube รองรับค้นหาชื่อเพลงหรือวาง URL ถ้าบอทกำลังเล่นอยู่จะเพิ่มเข้าคิวอัตโนมัติ ต้องอยู่ในช่องเสียงก่อน
ทุกคน
⏭️
/skip
ข้ามเพลงที่กำลังเล่นอยู่ไปยังเพลงถัดไปในคิว ถ้าไม่มีเพลงถัดไปบอทจะออกจากช่องเสียง
ทุกคน
⏸️
/pause
หยุดเพลงชั่วคราว ต้องเล่นเพลงอยู่ก่อนถึงจะใช้ได้ สามารถกลับมาเล่นต่อด้วย /resume
ทุกคน
▶️
/resume
เล่นเพลงต่อจากที่หยุดชั่วคราวไว้ด้วย /pause ต้องอยู่ในสถานะ paused เท่านั้น
ทุกคน
⏹️
/stop
หยุดเพลง ล้างคิวทั้งหมด และบอทออกจากช่องเสียงทันที
ทุกคน
💡

คำสั่งทั้งหมดเป็น Slash Commands ใช้งานได้ทุกคนในเซิร์ฟ ไม่จำกัดสิทธิ์ ต้องเข้าช่อง Voice ก่อนสั่ง /play บอทใช้ yt-dlp + FFmpeg สตรีมเสียงจาก YouTube
